# בטן רגועה 🌸

אפליקציית PWA אישית בעברית (RTL) לניהול עדין של מעי רגיז אצל מתבגרת, עם דגש על כאב ונפיחות.
כל הנתונים נשמרים **רק על המכשיר** (localStorage + IndexedDB). אין שרת, אין חשבון, אין מעקב.

## מה יש בפנים

- **היומן שלי**: ציר זמן יומי מקסים (ארוחות, הרגשות עם עוצמה והקשר, הקלטות קול, פתקים), ממוין אוטומטית לפי ימים ושעות.
- **ארוחות**: בנק רעיונות בגישה עדינה לבטן (בוקר, צהריים, ערב, נשנושים, קינוחים, בית ספר), עם חלופות לכל ארוחה, הגרלה 🎲, ורישום ליומן בלחיצה אחת.
- **טיפים**: הרגלים שעוזרים לכאב ונפיחות + תרגיל נשימת בטן אינטראקטיבי (4-4-6).
- **סיכום**: סטטיסטיקה של 14 יום, שיתוף סיכום טקסט לדיאטנית/הורים, גיבוי ושחזור.

## פריסה ב-GitHub Pages

1. יוצרים ריפו חדש (למשל `beten-ragua`), אפשר פרטי או ציבורי.
2. מעלים את **כל** הקבצים מהתיקייה הזו כפי שהם (כולל תיקיית `icons`).
3. Settings → Pages → Source: `Deploy from a branch` → Branch: `main` → `/ (root)` → Save.
4. אחרי דקה-שתיים הכתובת תהיה:
   `https://<username>.github.io/beten-ragua/`

## התקנה על האייפון שלה

1. פותחים את הכתובת **בספארי** (חובה ספארי).
2. כפתור שיתוף (הריבוע עם החץ) → **הוספה למסך הבית**.
3. האפליקציה תיפתח במסך מלא עם האייקון שלה.

### הרשאת מיקרופון (להקלטות)

בלחיצה ראשונה על 🎙️ ספארי יבקש הרשאת מיקרופון, מאשרים פעם אחת.
אם סורב בטעות: הגדרות → ספארי → מיקרופון, או הגדרות → אפליקציית "בטן רגועה" (iOS 17+).

## הערות טכניות

- ההקלטות נשמרות כ-Blob ב-IndexedDB על המכשיר בלבד ואינן נכללות בסיכום הטקסט.
- ה-Service Worker טוען את האפליקציה גם בלי אינטרנט; עדכון קוד באתר מגיע אוטומטית בפתיחה הבאה עם רשת.
- גיבוי: כפתור "גיבוי היומן" מוריד קובץ JSON. מומלץ לגבות מדי פעם, כי מחיקת נתוני ספארי מוחקת גם את היומן.
- עדכון גרסה: אם משנים את `index.html`, מומלץ להעלות את מספר הגרסה ב-`sw.js` (`beten-ragua-v1` → `v2`).
